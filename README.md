# G-EduShell
A minimalist reverse shell trojan for educational purposes, co-authored with Google Gemini.
# Build
Use MSVC build tool:
```
cl /utf-8  shell.c /link /SUBSYSTEM:WINDOWS ws2_32.lib
```
